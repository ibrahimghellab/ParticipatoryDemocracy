Documentation de l'API

API pour la gestion des groupes et des utilisateurs

  • GET
  • POST
  • PUT
  • DELETE

GET Requests

  • /groupe/:id - Récupère les informations d'un groupe
  • /user/:id - Récupère les informations d'un utilisateur
  • /groupes - Liste de tous les groupes
  • /user/:id/groupes - Liste des groupes d'un utilisateur
  • /users - Liste de tous les utilisateurs
  • /groupe/:id/membres - Liste des membres d'un groupe
  • /groupe/:id/membre/:id - Détails d'un membre d'un groupe

POST Requests

  • /groupe - Créer un nouveau groupe
  • /user - Créer un nouvel utilisateur
  • /vote - Soumettre un vote

PUT Requests

  • /groupe/:id - Modifier un groupe existant
  • /user/:id - Modifier un utilisateur existant

DELETE Requests

  • /groupe/:id - Supprimer un groupe
  • /user/:id - Supprimer un utilisateur
